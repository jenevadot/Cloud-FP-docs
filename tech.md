## Stack Tecnológico

## Target Platform
- Os: Linux Ubuntu server

### Frontend
- **Framework:** React 18+ con TypeScript
- **Build Tool:** Vite
- **Estilos:** Tailwind CSS
- **Gestión de Estado:**
  - TanStack Query (React Query) para estado del servidor
  - Zustand para estado global (autenticación)
- **Enrutamiento:** React Router v6
- **Formularios:** React Hook Form + Zod
- **Fechas:** date-fns
- **HTTP Client:** Axios

### Backend
- **Framework:** FastAPI (Python 3.11+)
- **ORM:** SQLAlchemy 2.0 (async)
- **Validación:** Pydantic v2
- **Autenticación:** JWT + bcrypt

### Base de Datos & Caché
- **Base de datos:** PostgreSQL 15+

### Infraestructura AWS
- **Base de datos:** AWS RDS (PostgreSQL)
- **Backend:** AWS ECS 
- **Frontend:** AWS ECS  
- **Almacenamiento:** AWS S3 (logos de negocios, archivos)
- **Orquestación:** AWS ECS con Application Load Balancer
- **Monitoreo:** AWS CloudWatch
- **Emails:** AWS SES (Simple Email Service)
